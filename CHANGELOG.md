# Changelog

Alle relevanten Änderungen an **Redactly** werden in diesem Dokument erfasst.

## [0.2.0] – 2025-09-06
### Added
- Editor mit stabiler Platzhalter-Hervorhebung
- Drei-Spalten-Layout (Editor, Mapping, Preview)
- Export als JSON und Text
- Session-Management für mehrere Projekte
- Windows & macOS Build-Skripte

### Changed
- UI-Polish für TitleBar und Header
- Platzhalter klarer markiert (Farbcodierung)

## [0.1.0] – 2025-08-01
### Added
- Erste Beta-Version mit Basisfunktionen:
    - Text anonymisieren & rückumwandeln
    - Einfaches Mapping
    - Copy-to-Clipboard
